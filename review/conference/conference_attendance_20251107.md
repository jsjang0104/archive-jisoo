## 2025년 언어연구소/언어공학연구소/세미오시스 연구센터 가을공동학술대회 참석

### 주제: "언어 다양성과 인공지능: 미래 언어학의 도전"

#### 강어진: "다국어 언어 모델의 사실적 지식 처리 연구: 언어와 사고 관점에서"

**1. 발표 내용:** (이해한 범위까지만...)

연구 주제: prompt configurations for exploring how language allignment affects factual knowledge-based response (언어 정렬(입력 언어와 모델의 응답 언어 간의 일관성)이 사실적 지식 기반 응답에 미치는 영향을 탐색하기 위한 프롬프트 구성)

language 분류: input / thought / output

knowlege를 지식 발생(source)와 지식 사용(context)에 따라 4종류로 분류
- Universal -> Universal
    - Science, ...
- Universal -> Localized
    - Bias?
    - ex) 흑인은 달리기를 잘 해.
- Localized -> Universal
    - Culture?
    - ex) 2024년 한국의 1인당 GDP는?
- Localized -> Localized
    - Culter, Language, ...
    - ex) "손이 크다"는 무슨 뜻인가요?


**2. 들었던 생각**
- 발표자님은 실험 대상 언어를 영어/중국어/아랍어로 하셨는데, 같은 한자 문화권인 한국어-중국어 그리고 같은 어족인 영어-독일어 등으로 묶어서 실험했을 때의 결과가 궁금하다.
- 집에 와서 자료를 다시 봤더니 모두 흑백이라 결과 해석이 힘들어서 본 문서에서 통채로 생략하였다 ㅜㅜ 다음부터는 이런 상황을 대비해 필기를 해두어야겠다.
- 학부생 수준으로 학술대회에 가서 내용을 100% 이해하진 못하더라도, 이러한 환경에 스스로를 자주 노출시키는 것이 중요한 것 같다.
- 발표 스킬 연마에 대한 중요성을 더 깨닫게 되었다. 연구자로서 발표할 일을 피할 수는 없으니까.
- 1년 안에 학부 연구생으로서 내놓을만한 결과물을 만들어 학회에 참석하는 것을 목표로 하고 싶다. (포스터 발표, 그냥 발표 등)
